# IoTNintendoSwitch
Arduinoによる任天堂スイッチの自動操作

Arduino Leonardoを利用して、任天堂スイッチを自動操作するコードです

今回はモンスターハンターライズというソフトで役に立つコードを書きました

使用ライブラリ：https://github.com/interimadd/NintendoSwitchControll

大学でのIoTのセンサー活用のゼミにて、発展編として後半のスライドで簡易的に発表
